# Water Quality AI System

This project predicts water potability using Machine Learning and enhances results using AI-based explanations.

## Features
- Water quality prediction using Random Forest
- Streamlit dashboard interface
- AI-based explanation (LLM with fallback)
- Risk alerts and recommendations
- Docker containerized deployment

## Technologies Used
- Python
- Scikit-learn
- Streamlit
- OpenAI API (optional)
- Docker

## Run Using Docker

docker pull riavarsha33/water-quality-ai  
docker run -p 8501:8501 riavarsha33/water-quality-ai  

Open in browser:  
http://localhost:8501  

## SDG Goal
SDG 6: Clean Water and Sanitation

## Project Description
This system helps in determining whether water is safe for drinking using machine learning models. It also provides intelligent explanations and suggestions.
